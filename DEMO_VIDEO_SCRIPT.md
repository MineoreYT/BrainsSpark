## üìù RECORDING TIPS

### Before Recording:
1. **Prepare demo data:**
   - Create teacher account: demo@teacher.com
   - Create student account: demo@student.com
   - Create sample class: "Math 101"
   - Create 2-3 sample quizzes
   - Have some quiz results ready

2. **Clean up browser:**
   - Close unnecessary tabs
   - Use incognito mode
   - Zoom to 90% for better visibility
   - Hide bookmarks bar

3. **Test audio:**
   - Use good microphone
   - Record in quiet room
   - Test levels before full recording

### During Recording:
1. **Screen recording settings:**
   - 1920x1080 resolution
   - 30 FPS
   - Record system audio OFF
   - Cursor highlighting ON

2. **Pacing:**
   - Speak clearly and enthusiastically
   - Pause 1 second between sections
   - Don't rush - let viewers see the UI

3. **Mouse movements:**
   - Move cursor smoothly
   - Hover over buttons before clicking
   - Circle important elements

### After Recording:
1. **Editing:**
   - Speed up repetitive actions (2x speed)
   - Add smooth transitions between sections
   - Add background music (soft, upbeat)
   - Add text overlays for key features

2. **Polish:**
   - Color correct if needed
   - Add subtle zoom-ins on important parts
   - Add sound effects for clicks (optional)
   - Export in 1080p
